About DICOMFortable

DICOMFortable is a DICOM 3D bone metastasis tool developed using the Python Flask framework.

Note: I had to remove the toggling of the buttons since they were not functioning properly on the server. I highlighted them in the current manual.

    TIPS

  • The HELP button will redirect you to this help page.
  • Please check your email for the access key or email me if your access key is not working.
  • Press RELOAD if the background color of the RELOAD button is orange. one reload should be enough
  • Window and level adjustments will be set for the current patient only.
  • Window/Level are not adjustable in the sagittal and coronal views yet.
  • The captured points will NOT be saved into the database if you do not press the SAVE button!
  • You can capture more points or remove points from the list and press the SAVE button to overwrite the previous list
  • The frames, scalings and views will reload if you press the reload button or go to the previous/next patient.
  • The program will NOT allow you to move to the next/previous patients if there are unsaved points.
  • reloading the frame will NOT remove the points that saved to the database.
  • Pressing the SAVE button will overwrite the points for the current patient only.
  • You need to use your access key to log into the app if you refresh your browser.
  • If you refresh the page only new patients (non labeled ones) will be loaded to the app.
  • Previously-labeled patient data will not be accessable if you refresh the page. I should manually remove that patient from the database if you want to have access to them.

    • How to Use

    Using your ACCESS KEY log into the DICOMFortable app.

    Once loaded, you will see a 3D view of DICOM files for the first patient.

    Press to fully load patients' data. The background color of the reload button will turn green if the patient's data is correctly loaded

    Use and buttons to adjust the window center and window width in the axial view.

    Use ZOOM IN () and ZOOM OUT() buttons to zoom in and zoom out the views. All three views will be magnified at the same time.

    Click and hold down the left mouse button to pan in each view. Other views will be adjusted accordingly.

    Use the mouse wheel scroll in any of the views to change slices in that view. Other views will be adjusted accordingly.

    Use the mouse double click in the any of the views to set the indicator position to the desired point in all 3 views.

    Once the indicator is in the center of the bone met in all three views, click the CAPTURE () button to capture the position of the bone met center. The bone met 3D pixel position will be added to the list on the left side of the window.

  • Note 1: Multiple bone met centers can be captured for each patient.

    • Pressing the UNDO () button will remove the last captured point from the list.

    Click on the SAVE () button to save the captured points. The button will be switched to the CHECKED BOX () icon, once the points are stored in the database.

    Use PREVIOUS () or NEXT () buttons to load the previous or next patient. Then, press RELOAD () button to reload the patient's metadata.

    By pressing the RELOAD () button at any stage, the setting will be reset and the captured points will be removed. The saved points will NOT be removed if you RELOAD the page. {% endblock %}
